SVM PE Pixel Extraction Utility Program
</h1>
<p>
A software which can extract <b>Pixel arrays</b> from <b>1-bpp Monochrome Bitmaps</b> and transform those extracted pixel data into <b>ESC\POS Single-bit Rasters</b>.
</p>
